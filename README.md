# 🚗 Garage Management System API

A **Flask-based RESTful API** for managing a garage ecosystem including users, mechanics, garages, cars, services, spare parts, and image uploads using Cloudinary.

---

## 📌 Overview

This API powers a garage management platform where:

- Users can register and log in  
- Mechanics and garages can offer services  
- Cars and spare parts can be listed  
- Images are uploaded and managed via Cloudinary  

---

## 🛠 Tech Stack

- **Backend:** Flask, Flask-RESTful  
- **Database:** SQLite (SQLAlchemy ORM)  
- **Authentication:** JWT (Flask-JWT-Extended)  
- **Image Hosting:** Cloudinary  
- **Migrations:** Flask-Migrate  
- **CORS:** Flask-CORS
